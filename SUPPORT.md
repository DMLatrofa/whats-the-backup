# Support

## Getting Help

- Use GitHub Issues for bugs and feature requests.
- Use Discussions if the repository enables them for broader product or usage questions.
- Follow `SECURITY.md` for anything security-sensitive.

## What to Include

- Operating system
- App version or commit SHA
- Steps to reproduce
- Expected behavior
- Actual behavior
- Sanitized screenshots or logs only

## What Not to Include

- Real WhatsApp exports
- Personal media
- Private chat content
- Secrets or tokens
